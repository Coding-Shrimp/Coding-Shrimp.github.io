记录

这个错是在初始化的时候报的错误
BMKMapView* mapView = [[BMKMapView alloc]initWithFrame:CGRectMake(0, 0, 320, 480)];
遇到这个错，做个记录，加深印象

  • 错误

解决方法

-方法1 手动更改我们本地的链接设置。
Build Settings–>Apple LLVM 7.1 - Language - C++ –> C++ Standard Library，设置为：libstdc++(GNU C++ standard library)

-方法2 添加库
Build Phases –> Link Binary With Libraries里面增加库：libstdc++.6.0.9.tdb
